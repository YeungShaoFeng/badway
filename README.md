# ***badway***
- **A encryption and decryption with pi.**

# **compile**
> **cmake . && make**

# **run**
- **encryption**
> **badway -e -i \<inputFile\> -o \<outputFile\> -p \<password\>**
- **decryption**
> **badway -d -i \<inputFile\> -o \<outputFile\> -p \<password\>**

# **test**
- **linux**
> **./test.sh**
- **windows**
> **.\\test.bat**
